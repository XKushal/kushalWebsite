StrengthLens is an iOS and Android app built with React Native and Expo. The project combines a modern mobile stack with an AI-first product direction, using Node.js and TypeScript on the backend and Supabase with PostgreSQL for authentication and data storage.

## What I Built

I built the application around a mobile-first workflow and explored how generative AI can make the product more useful through retrieval-augmented generation and Model Context Protocol integrations that connect tools and project context to large language models.

## Tech

- React Native / Expo
- Node.js
- TypeScript
- Supabase
- PostgreSQL
- Generative AI, RAG, and MCP
